# GUIDE
## Welcome : 
We will take you from start to finish to get your own OpenHPC cluster up and running with GUI monitoring, WITH ONLY Open-source tools. 

### DISCLAMER : 
We used only hardware for this project and guide, however other methods can be followed for VMs or virtualized clusters. 

### Materials required : 
 - Atleast 2 computers (1 admin and 1 compute node)
 - Ethernet Switch with PortFast (or equivilant for switch brand/vendor to bypass Spanning Tree Protocol (STP))
 - Ethernet cables
 - PCIE ethernet card (if your motherboard has only a single ethernet port)
 - General PC peripherals (monitor, mouse, keyboard)
 -  A will to live (very important)

### Materials used for the project/guide :  
 - 5 ThinkCentre E73
 - Cisco Catalyst 3560cx
 - CAT 5e or CAT 6 ethernet cables
 - TP link TG-3468
